# Comandos Básicos

## Comandos Estruturais

- git config --list = traz a estrutura configurada na máquina;
- git config --global --unset user.email = remover configuração user.email;
- git status = demostra se há alguma alteração ou pendência;

## Passos de envio

1. git add *  = adiciona pastas e arquivos
2. git commit -m "detalhamento alteração"   = grava alterações para envio
3. git push origin master    = envia git
